\subsection{circle.py}
This program finds the area and perimeter of a circle using mathematical formulas: \\
\\
Circle area: $$S = \pi R^{2}$$
Circle Perimeter: $$P = 2\pi R$$
Here is the source code of the program:

\begin{minted} {python}
import math
\end{minted}
\textit{A library in Python that loads a standard math module that provides many mathematical functions and constants}

\begin{minted} {python}
def area(r):
    return math.pi * r * r
\end{minted}
\textit{The function takes one argument r, which is the radius of the circle and uses the constant $\pi$ from the math module to calculate the area of a circle using the formula: $S = \pi R^{2}$}

\begin{minted} {python}
def perimeter(r):
    return 2 * math.pi * r
\end{minted}
\textit{The function takes one argument r, which is the radius of the circle and uses the constant $\pi$ from the math module to calculate the perimeter of a circle using the formula: $P = 2\pi R$}

\\

\begin{minted} {python}
def area(a):

\begin{minted} {python} 
def perimeter(a):
    return 4 * a
\end{minted}
\textit{The function takes one argument a, which is the length of the side of the square and the square's perimeter, calculated by the formula $P = 4a$}
